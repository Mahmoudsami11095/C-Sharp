# C# Course Assignments

A comprehensive collection of C# assignments tracking progress through the backend course.

## 🚀 Course Overview

This repository contains assignments and hands-on exercises for learning C# fundamentals, including:
- Variables, scope, and memory management
- Exception handling
- Object-oriented programming concepts
- Type system basics

## 📂 Repository Structure

This repository uses a branch-per-assignment strategy:
- `main`: Contains the repository documentation and base files.
- `assignment-1`: Contains the code for Assignment 1.
- `assignment-2`: Contains the files for Assignment 2.

## 🛠 Getting Started

### Prerequisites
- .NET 9 SDK (or later)
- Visual Studio 2022 or Visual Studio Code
- Git

### Running an Assignment
1. Switch to the branch for the assignment you want to review:
   ```bash
   git checkout assignment-1
   ```
2. Navigate to the assignment folder and run the project:
   ```bash
   cd Assignment_01/Assignment
   dotnet run
   ```
