### Assignment 1: C# Fundamentals (20 Questions)
**Topics Covered:**
- Regions and code organization
- Variable declarations (explicit vs implicit)
- Constants and scope
- Class-level vs method-level scope
- Block-level scope and variable lifetime
- Garbage collection
- Variable shadowing
- Naming rules and conventions
- Error types (syntax, runtime, logical)
- Exception handling
- try-catch-finally blocks
- Built-in exceptions
- Multiple catch blocks
- throw vs throw ex
- Stack and heap memory
- Value types vs reference types
- System.Object and inheritance
