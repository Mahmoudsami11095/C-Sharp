# Assignment 1: C# Fundamentals

## 📝 Overview
This is the first assignment for the C# course, covering the foundational concepts of the C# programming language. It consists of 20 questions exploring theoretical concepts and practical syntax.

## 🎯 Topics Covered
- Regions and code organization
- Variable declarations (explicit vs implicit)
- Constants and scope (Class-level, method-level, block-level)
- Garbage collection and variable lifetime
- Variable shadowing
- Naming rules and conventions
- Error types (syntax, runtime, logical)
- Exception handling (`try-catch-finally` blocks, built-in exceptions)
- `throw` vs `throw ex`
- Stack and heap memory
- Value types vs reference types
- System.Object and inheritance

## 🚀 How to Run
Navigate to the project directory and execute the following commands:
```bash
cd Assignment
dotnet run
```

- Each question includes detailed comments explaining the concept or solution.
